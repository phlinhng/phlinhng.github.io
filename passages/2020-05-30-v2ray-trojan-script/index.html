V2Ray TCP+TLS+Web , V2Ray WSS+CDN , Trojan-Go 三合一脚本 科学上网 一键脚本

脚本来到 v1.1.3 版，现在己经进化成 V2Ray TCP+TLS+Web , V2Ray WSS+CDN , Trojan-go 三合一脚本，三种梯子可以共存。小孩子才做选择，大人全都要。

使用

特色

  • 支持的配置 ( 可共存 )
    • V2Ray TCP+TLS+Web
    • V2Ray WSS+Web+CDN
    • Trojan-Go
  • 默认采用 Domain Socket
  • 自动更新 TLS 证书
  • 自动更新最新 geoip.dat, geosite.dat
  • 自动校正 VPS 时间
  • 设置电报代理 ( 带 TLS 混淆 )
  • 生成全客户端订阅链接
  • 集成常用 VPS 指令
    • 开启 swap、安装 TCP 加速内核、测速脚本、卸载阿里云盾

支持系统

  • Debian 9+
  • Ubuntu 16.04+
  • 推荐使用 Ubuntu 18.04

项目地址

  • 主页：https://github.com/phlinhng/v2ray-tcp-tls-web/wiki
  • 源码：https://github.com/phlinhng/v2ray-tcp-tls-web

Keywords

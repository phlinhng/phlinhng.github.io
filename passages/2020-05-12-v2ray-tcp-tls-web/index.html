v2Ray TCP+TLS+WEB 一键脚本 2020比WS+TLS更快的最新科学上网方式

全网第一个采用Domainsocket的 v2Ray TCP+TLS+WEB 一键脚本。
项目地址: https://github.com/phlinhng/v2ray-tcp-tls-web/wiki

一开始为了自用写了这个脚本，后来在一个issue的建议下开始了增加功能的不归路。TCP+TLS+WEB是由WS+TLS+WEB改良的新v2Ray配置组合，拥有比WSS更低的延迟和同样的伪装能力，理论上速度与Trojan-GFW一致。本脚本使用TLS分流器进行主要实现、Caddy in Docker做80 http到443 https的重定向，并集成了生成订阅链接、设置电报代理、安装加速内核等功能。

v2Ray TCP+TLS+Web 一键脚本

特色

  • 全网第一个 TCP+TLS+Web 一键脚本
  • 默认采用Domain Socket
    • 是的，这是一个 v2Ray TCP TLS Web DS 一键脚本
  • 设置电报代理
  • 生成全客户端订阅链接
  • 集成常用VPS指令: 开启swap、安装TCP加速内核、测速脚本、卸载阿里云盾

功能简介

v2script

1) 安装TCP+TLS+WEB: 就是安装，如果域名解析打错或是想要修改可以執行此行重新生成配置。
2) 更新v2Ray-core: 使用v2Ray官方脚本更新v2Ray-core二进制文件
3) 卸载TCP+TLS+WEB: 完全卸载v2Ray-core, tls-shunt-prxoy, docker containers并删除所有依赖与配置文件。
4) 显示vmess链接: 显示v2rayN格式的vmess链接，适用v2RayNG, Shadowrocket, Pharos Pro
5) 订阅管理: 进入订阅管理工具
6) 设置电报代理: 设置电报代理
7) VPS工具: 进入VPS工具箱

v2sub

1) 生成订阅: 生成v2rayN格式的订阅链接，托管在v2Ray自身VPS的网站上，您的节点不会被偷
2) 更新订阅: 修改现有订阅链接的名称，也适用于手动修改/etc/v2ray/config.json后更新订阅(只支持读取本脚本所建立的预设格式)
3) 显示订阅: 显示订阅连接 (生成全平台订阅连接需要安装API，若未安装会询问是否要安装API)

VPS工具箱

1) 安装加速: 安装@chiakge/Linux-NetSpeed提供的四合一加速脚本，感谢加速脚本作者。
2) 设置Swap: 在预设没有开启swap的VPS上开启swap，对增加谷歌云稳定性与让谷歌云不再提示升配置有奇效
3) 卸载阿里云盾: 卸载阿里云安骑士相关组件，仅在阿里云VPS上有效
4) 性能测试: 使用@LemonBench/LemonBench提供的测试脚本进行性能测试，感谢测试脚本作者。

創站

其實從 2019 年開始一直有自己建 blog 的念頭，因為那時候發現大陸厲害的程序員都喜歡自己架站，不過那時候寫代碼的功力還很菜就一直懶得弄。

就在上個月（2020年 1 月）中，我用了3年的知乎帳號被無預警停用了，200 多篇收藏、不知道多少回答和 5 篇專欄文章隨之消失，這時才了解為什麼大陸程序員要自己建博客自己架站保存內容的重要性。雖然目前我發表過的內容大部分是寫給自己看的備忘錄，好歹也是我的勞動成果啊！有個自己的博客至少不會因不可抗力因素說沒就沒了。

架構採用很流行的 Hexo + Github Pages + 自訂域名 的方式，雖然國內訪問github比較慢但至少不會被牆，以保存筆記的用途來說應該夠用了。主題是 github/dongyuanxin 大佬寫的 Art Design 主題 theme-ad。

創站日期為 2020-02-26，設定成 2020-02-20 只是看著順眼。

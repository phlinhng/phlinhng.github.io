解决 Redis 服務 Can't open PID file

更新 v2board 至最新 dev 版

修改 Redis.service

若更新後無法進入面版，需要修改 redis.service

在 [Service] 加入 ExecStartPost=/bin/sh -c "echo $MAINPID > /var/run/redis/redis.pid" 傳遞 PID 參數給 redis，即可避免此 bug

保存設置重啟

v2board 問題排查思路

  1. v2board 路徑是否設置遞歸 755 權限
  2. php7.x-fpm.service、redis-server.service 是否正常運行
  3. PM2 隊列是否正常啟動
  4. composer 組件是否均正確安裝
